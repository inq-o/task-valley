# Task Valley

Task Valley는 현실의 일일 과제를 완료하면 어두운 도트 마을이 회복되는 작은 자기계발 RPG입니다.

## MVP 방향

- 기술 스택: Next.js, TypeScript, Tailwind CSS
- 저장 방식: v0.1은 localStorage, 이후 DB 확장
- 핵심 루프: 오늘의 4개 과제 생성 → 완료 → 마을 구역에 에너지 전달 → 마을 회복 확인
- AI 협업 역할:
  - Claude Code: 기획자 및 리뷰어
  - Antigravity / Nano Banana: 디자이너 및 비주얼 프로토타이퍼
  - Codex: 개발자
