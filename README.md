# Task Valley

Task Valley is a small self-improvement RPG where real-life daily tasks restore a dark pixel village.

## MVP Direction

- Stack: Next.js, TypeScript, Tailwind CSS
- Persistence: localStorage first, database later
- Core loop: create 4 daily tasks, complete them, send energy to town zones, and watch the village recover
- AI collaboration roles:
  - Claude Code: planner and reviewer
  - Antigravity / Nano Banana: designer and visual prototyper
  - Codex: implementation engineer
